# |EMULinux|
EMULinux is based of the JSLinux project.

# Features:

1. Networking Support via Tailscale
2. More advanced File Systems
4. Diffrent Distros
3. Use your own distro (Made By Distro Packager)
4. Distro Packager
5. Silent Distro (Basically linux running without you knowing)

# About
EMULinux was made with the idea to have an more advanced emualtor for the web. EMULinux can run also in the backround linux distros so websites can be more advanced. EMULinux works on the client side too so no server overhead (execpt normal http/https stuff).
